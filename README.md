# 📑 My Simple Todo List
> **학습 목표:** HTML/CSS/JS의 기초인 CRUD 구조 이해 및 깃허브 배포 경험

---

## 🚀 프로젝트 소개
사용자가 오늘의 할 일을 기록하고 관리할 수 있는 웹 어플리케이션입니다. 
깔끔한 UI와 반응형 디자인을 적용하여 모바일에서도 편리하게 사용할 수 있습니다.

### 🔗 바로가기
* **실행 웹사이트:** [👉 여기를 클릭하여 접속](https://donjy2040.github.io/todo/index.html)
* **개발 기록 (기획서):** [상세 기획서 보기](./todo.md)

---

## ✨ 핵심 기능
1. **할 일 추가**: 입력창에 할 일을 적고 '추가' 버튼 클릭
2. **리스트 삭제**: 휴지통 아이콘 또는 삭제 버튼으로 항목 제거
3. **완료 토글**: 할 일을 클릭하여 완료 상태(취소선)로 변경
